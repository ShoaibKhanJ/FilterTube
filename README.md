# 🎯 FilterTube

**Take Control of Your YouTube Feed.**

FilterTube is a Chrome Extension that helps users personalize their YouTube experience by filtering unwanted content, reducing distractions, and prioritizing videos that match their interests.

## ✨ Features

### 🔍 Keyword Filtering

Show videos that match your preferred topics.

Examples:

* AI
* Python
* Web Development
* Startups

### 🚫 Blocked Keywords

Hide or blur videos containing unwanted topics.

Examples:

* Gaming
* Pranks
* Celebrity Drama
* Clickbait

### 🎥 Hide YouTube Shorts

Remove Shorts from:

* Home Feed
* Recommendations
* Shorts Sections

### 🎯 Focus Mode

Reduce distractions by filtering irrelevant content and keeping your feed aligned with your goals.

### ⚙️ Flexible Display Modes

Choose how unwanted videos are handled:

* Blur Mode

  * Blur thumbnails and titles
  * Disable interaction
  * Reduce distractions

* Hide Mode

  * Completely remove irrelevant videos

### 📺 Channel Whitelist

Always allow content from selected channels regardless of filtering rules.

## 🚀 Installation

### Development

Clone the repository:

```bash
git clone https://github.com/your-username/filtertube.git
cd filtertube
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build production version:

```bash
npm run build
```

## 🎯 Project Vision

FilterTube aims to transform YouTube into a personalized learning and productivity platform by giving users full control over the content they consume.

Instead of letting algorithms decide what appears in your feed, FilterTube empowers users to define their own viewing experience.
